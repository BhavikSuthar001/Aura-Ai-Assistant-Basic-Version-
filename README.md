# Aura-Ai-Assistant-Basic-Version-
Aura Ai Assistance is An Basic Prototype of Ai Assistance which give answer of question and have some basic features like open any app or check weather.

# Feutures
1 - Uses Gemini Ai to give Answer of Quetion.
2 - Open Random app and some selected website.
3 - Set Screen Brightness 
4 - It can gives Battery Percentage of your laptop
5 - Can give info of whether in your city
