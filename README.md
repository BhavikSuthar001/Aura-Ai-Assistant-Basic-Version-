# Aura-Ai-Assistant-Basic-Version-
Aura Ai Assistance is An Basic Prototype of Ai Assistance have some basic features like open any app or check weather.

# Feutures
1. Uses Gemini AI to give answers to questions. (Currently unavailable due to some reason)
2. Opens random apps and selected websites.
3. Sets screen brightness.
4. Shows the battery percentage of your laptop.
5. Gives weather information for your city.

# Installation
Step 1 : Clone The Repositary

* git clone https://github.com/BhavikSuthar001/Aura-Ai-Assistant-Basic-Version-.git

Step 2 : Create Virtual Environment 

* python -m venv venv
* venv\Scripts\activate

Step 3 : Install required packages

* pip install -r requirements.txt

(If you don’t have a requirements.txt, tell me and I’ll generate it.)

# Usage
Run the assistant 
python main.py

* For initiate aura : 
Say "hello" after the assistant prints “Listening…”.

* After your task done say "shutdown" to stop assistant or say "hello" to use it again
